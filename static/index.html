Core Workflow:

  1. Setup Tunnel - Create ngrok tunnel for external access
  2. Install Client - Deploy one-click install script to target machines
  3. Submit Commands - Send commands to registered clients
  4. Monitor Results - Track execution status and retrieve outputs
